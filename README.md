## Deployment

### GitHub Pages

1. Push the repository contents to a branch (e.g. `main`)
2. Enable **Pages** in repository Settings → Source: `main` / `/ (root)`
3. Access via `https://<user>.github.io/<repo>/`

### Local development

```bash
python -m http.server 8080   # Python 3
npx serve .                  # Node.js
```
